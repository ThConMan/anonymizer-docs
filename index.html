Anonymizer

Stealth kills shouldn't dox the killer. Anonymizer redacts a fully-invisible player's name everywhere it shows — so going invisible actually means going dark.

Paper 26.1+ Java 21+ Premium · closed source
Get it Documentation

The idea

On competitive servers, going invisible still shouts your name the moment you strike.

Drink an invisibility potion and you look hidden — but the moment you land a kill, your name is plastered across chat, the death message, the tab list, and the floating nametag above your head. Everyone instantly knows exactly who's stalking them. Anonymizer fixes that: while a player is fully invisible, their name is replaced with a flickering REDACTED block — so a stealth kill stays anonymous.

How it works

Every place your name can leak — covered.

Redacted name

A hidden player's name becomes a flickering bold-white block (Minecraft's &k obfuscation). Tune the flicker, the glyph, and the length — or switch it to a clean, solid bar.

Chat

Redacts the sender's name in chat. It runs last (MONITOR priority) and replaces by content, so it survives other chat plugins' formatting — even DragonReign's gradient titles.

Death messages

Scrubs the real name of a hidden killer — or a hidden victim — out of the death broadcast, so the server can't reverse-engineer who was involved.

Tab list

Default tab-mode: hide-skin keeps the player listed but strips their skin, so the head shows a generic default — the real skin is never revealed. Other modes: name-only, remove, off.

Nametag

Hides the floating in-world username (via a scoreboard team) so a hidden player's name can't be read through walls or across the map.

Plugin hooks

Auto-detected by reflection — no hard dependencies. EssentialsX nicknames are redacted in /list, /seen, /near, and /whois; CMI nicknames and VentureChat are covered too. ChatControl, DeluxeChat, and CarbonChat get a logged config tip.

Smart trigger

Anonymizes whenever a player has the Invisibility effect. Want true ghosts only? Require no armor and empty hands so only a fully invisible player goes dark.

Live preview & status

Run /anonymizer preview to see exactly what your redacted name looks like, and /anonymizer status to check whether a player is currently being anonymized and why.

How it plays out

Invisible means invisible — including your name.

1

Go invisible

A player drinks an invisibility potion (and, if you require it, drops their armor and weapons for a true ghost).

2

Go dark

Their name is redacted everywhere it would otherwise appear — chat, death messages, tab, and the floating nametag.

3

Strike unseen

Land the kill. The death message and chat name out only a flickering block — no one learns who's hunting them.
